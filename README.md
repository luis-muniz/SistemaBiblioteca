# SistemaBiblioteca
Um sistema feito na disciplina de banco de dados para gerenciar um biblioteca, onde o aluno pode fazer empréstimos de livros e a funcionária pode autorizar os empréstimos.

# Ferramentas
1. MySql
2. Materialize
3. Apache
4. Jquery
5. Mpdf
